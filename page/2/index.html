阅读笔记

《猜测和偏见：何帆阅读笔记》

-人类自称是地球的主宰，历史似乎只是对人类活动的记录，但如果你从太空远眺地球，人类就渺小得根本无法看见。距离地球再近一些，比如当你坐在飞机上俯瞰大地的时候 ，就会发现人类仅仅聚居在地球上很小的一块块土地上：凡是有人居住的地方，就像大地长出来的癣疥。技术的发展，使得人们狂妄地认为人类可以摆脱自然的束缚，但至今为止，地理仍然能够在很大程度上约束人类的行为。总有些地方人类无法居住，总有些地方部队无法交战，总有些地方难以形成城市，总有些地方易于遭受灾害。
-由于人必须生活在人群之中，这就有了社会，也就有了规范社会的宗教和道德。不同时代的道德会有极大的差异，但不管是什么样的道德，都必须有道德： 贞操可以是一种道德，性开放也可以是一种道德；杀人可以是一种道德，护生也可以是一种道德。道德是一种规范社会行为的工具，是一个社会里人人必须穿上的内裤。
-同时追求自由和平等，是一种虚妄：若有无节制的自由，必有不断扩大的不平等；若要人人平等，必然会限制某些人的自由。
-绝大部分历史是猜测，其余的部分则是偏见。
-西方世界的四个支柱，即民主、资本主义、法律和公民社会
-政治决策是混乱、仓促而多变的。很多决策不过是不同利益之间的妥协，各种不完美方案的拼凑。
-1968年是个动荡不安的年份。

写在前面

毕业之后，工作之前，决定开个个人博客，记录工作与生活中的思考

  • 本篇第一是为了测试搭建的博客框架，看看我一团糟的代码是否也能够搭建成一个简洁整齐的博客

  • 第二是想谈一谈这篇博客主要用于什么以及为什么建立这个个人博客

    人在不断前行的过程中，适当的反思与回顾是很重要的。之前也有过记日记 的习惯，不过日记这种载体，只有自己能够看到，当写得多了，会陷入一种自己的世界，慢慢变得固执，与世界脱节。而国内的社交网络的载体，更适合发布一些生活日常，所以慢慢的我现在在朋友圈都是以自黑为主，主要也是为了让大家感受到快乐，毕竟这种轻社交的目的也并不在于让大家思考，我们刷社交网络也是图个乐呵，很少会对一些问题产生深入的讨论。
    之前的公众号，或者知乎专栏等地方，倒是适合记录，但是我更想在更加自由，更缺乏管制的地方，来谈一谈新闻或者政治之类的话题，所以最终决定在github上搭建博客。大家如果能看，我就很高兴了，如果能够有一些反馈的话，那我就更加欢喜了。
    2018年8月8日 于杭州

运用hexo和github构建博客的方法

过几天入职的时候，又得换电脑了，现在记录一下搭建这个博客的历程，免得到时候忘了。作为一个没有代码基础的人，我写的文章应该会更加通俗易懂。相信不会写代码的读者在看了我的文章之后，也能够在一个晚上搞定自己的个人博客搭建工作。

  • 环境
  • 工具
  • 过程

环境

首先，先说环境。我的电脑系统是64位的win10，然后电脑一直有挂着VPN，不过我试过，本文提到的网页或者软件都是不需要翻墙就能够获取的，问题不大。

工具

  1. node.js
  2. visual studio code
  3. git
  4. github账号
  5. hexo

过程

第一步，可以先注册github的账号，百度搜索github就行
第二步，在github的个人主页下创建一个新的仓库
第三步，下载并安装git。然后在桌面上右键，点git bash here。输入ssh-keygen，回车。然后找到秘钥文件，用记事本打开，复制到github上的setting-ssh里面。
第四步，确定git和github已经连接。输入：
git init
git remote add origin http://(复制粘贴第二步建立的库的http网址）
git pull origin master
